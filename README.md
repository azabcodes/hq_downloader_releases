# 🐝 HQDownloader - Releases & Updates

This is the public distribution repository for **VidBee** Android Application releases. 

While the core source code of VidBee is kept private to protect intellectual property, this public repository is used by the application to:
1. Fetch and query update metadata.
2. Download signed release APKs directly to your device.

### How to Install
* Go to the [Releases](https://github.com/azabcodes/hq_downloader_releases/releases) page.
* Download the latest APK compatible with your device architecture.
* Open the downloaded APK on your Android device to install or update the app.

### Device Architecture Guidelines
* **arm64-v8a**: Most modern Android devices (64-bit).
* **armeabi-v7a**: Older Android devices (32-bit).
